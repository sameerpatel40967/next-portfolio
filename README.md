This is a production-ready starter portfolio built with Next.js 14, React and Tailwind CSS.
Features:
- Dark glassmorphism theme
- Framer Motion + AOS animations
- Typewriter effect
- Projects loaded from `data/projects.json`
- Certificates carousel
- Contact form integrated with EmailJS (replace the SERVICE_ID/TEMPLATE_ID/PUBLIC_KEY)
- Responsive and SEO-ready (meta in `app/layout.js`)

## Setup

1. Install dependencies:
```
npm install
```

2. Run dev:
```
npm run dev
```

3. Build:
```
npm run build
```

## EmailJS

Sign up at https://www.emailjs.com and replace the placeholders in `components/Contact.jsx`.

## Deploy

This project is ready for Vercel/Netlify. Just push to GitHub and import the repo.
